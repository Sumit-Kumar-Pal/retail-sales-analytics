# Retail Sales Analytics (Pandas + NumPy)

A complete end-to-end analytics project using the **Online Retail II Dataset**.

## Features
- Data cleaning & preprocessing  
- Monthly, weekly, daily sales analysis  
- Product performance ranking  
- Customer RFM segmentation  
- Cohort retention analysis  
- Demand forecasting using NumPy moving averages  

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset
Online Retail II: https://archive.ics.uci.edu/dataset/502/online+retail+ii

## How to run 
### One by one in below sequence.
- python scripts/clean_data.py
- python scripts/analyze_sales.py
- python scripts/rfm_analysis.py
- python scripts/cohort_analysis.py
- python scripts/forecast_sales.py
### Or run  only 
-Retail_Analytics.py
